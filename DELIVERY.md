# Delivery

This folder contains a working Next.js project rebuilt from the markdown handoff in the workspace.

## Included

- Premium landing page layout
- Hero section with bold typography and CTA actions
- Sticky canvas sequence driven by scroll
- Story overlays for the journey section
- Services showcase cards
- Footer with contact and brand references
- Tailwind, TypeScript, and GSAP setup
- Utility scripts for real or sample frame generation

## Notes

- The app runs without bundled JPEG frames by using a procedural canvas fallback.
- Add real frames to `public/frames` when you are ready to replace the preview.

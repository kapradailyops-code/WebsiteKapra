# Project Summary

## What Was Built

- Premium one-page marketing site for Kapra Web AI Makers Pvt. Ltd.
- Hero with strong typography, CTA actions, and a cinematic mood
- Sticky canvas sequence section tied to scroll
- Story-driven overlay content
- Services showcase for web, mobile, and AI offerings
- Footer with contact and brand references

## Structure

- `src/components`: UI sections and shared page building blocks
- `src/pages`: Next.js entry points
- `src/hooks`: reusable animation hooks
- `src/utils`: frame loading helpers
- `scripts`: video frame utilities

## Design Direction

- Primary dark: `#0A0A0A`
- Secondary dark: `#1A1A1A`
- Accent blue: `#0EA5E9`
- Large display typography and spacious negative space
- Glassmorphic panels and subtle grid atmosphere
